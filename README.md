# 横スクロールシューティングゲーム

## 概要
自動で進行するステージを舞台に、プレイヤーが自機を操作して敵を撃破し、立ちはだかるボスを倒してゲームクリアを目指す横スクロールシューティングゲームです。

## 遊び方（操作方法）
- **移動:** 上下左右の矢印キー (`↑`, `↓`, `←`, `→`) で自機を操作します。
- **ショット:** スペースキー (`Space`) で弾を発射します。弾は虹色に変化します。
- **残機:** プレイヤーは3つの残機を持っています。敵やボスの攻撃に当たると残機が減ります。残機が0になるとゲームオーバーです。
- **復活:** ダメージを受けて残機が減っても、その場ですぐに復活し、一時的に無敵時間が発生します。
- **ボス戦:** スコアを100点稼ぐごとにボスが出現します。ボスは倒すごとに発射する弾の数が増え、手強くなります。
- **ゲームクリア:** ボスを3体倒すとゲームクリアとなります。
- **デバッグ機能:** `C` キーを押すと、すぐに第3ボス戦までスキップできます。

## 工夫した点
- **視覚的に楽しいシューティング:** プレイヤーの弾が虹色に変化し、ゲームプレイに彩りを加えています。
- **バリエーション豊かな敵:** 3種類の異なる動きをする敵キャラクターが登場し、飽きさせない工夫をしています。
- **戦略的なボス戦:** ボスはレベルが上がるごとに攻撃が激しくなり（撃ってくる弾の数が増える）、戦略的な立ち回りが求められます。
- **豪華なゲームクリア演出:** ボスを3体倒した際には、派手なパーティクルエフェクトと共にゲームクリアを祝福します。
- **ユーザーフレンドリーな残機システム:** 残機が減っても即座にゲームオーバーにならず、無敵時間付きでその場に復活するため、初心者でも安心してプレイできます。
- **開発しやすい構造:** JavaScriptのクラスベースの構造を採用し、各要素が独立して管理しやすいよう設計しています。
